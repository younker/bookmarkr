# Findr

Fuzzy finder for chrome bookmarks.

# How To Find

Once installed, hit `ctrl+b` (or `cmd+b`) and start typing.

# Results Navigation

- `cnrl+p`: Scorll up
- `cnrl+n`: Scroll down
- `return`: Open bookmark in a new tab
